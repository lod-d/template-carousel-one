# Carousel

Création d'un carousel en JavaScript, HTML et CSS.

## Fonctionnalités

* le carousel est composé de 4 images
* les images sont enregistrées dans le dossier `img`
* la classe `active` est appliquée à l'image actuellement affichée
* les boutons "précédent" et "suivant" permettent de naviguer dans le carousel
* le carousel est responsive et s'adapte à la taille de la fenêtre

## Installation

* télécharger le dossier `carousel` et le placer dans le dossier de votre choix
* ouvrir le fichier `index.html` dans un navigateur web

## Technologies utilisées

* HTML5
* CSS3
* JavaScript
